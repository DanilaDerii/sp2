pack.json - pack metadata

{
  "pack_id": "...",
  "title": "...",
  "version": "v1",
  "description": "...",
  "embedding_model": "all-minilm:latest",
  "embedding_dim": 384,
  "tutor_mode": "debug",
  "default_top_k": 5,
  "created_at": "...",
  "builder_version": "v1-prototype"
}

chunks.json - array of chunk records

[
  {
    "chunk_id": "...",
    "source_id": "...",
    "source_type": "pdf",
    "source_title": "...",
    "text": "...",
    "chunk_index": 0,
    "page": 1,
    "section": null,
    "topic": null,
    "char_count": 900
  }
]


vectors.npy - numpy matrix of embedding vectors

vectors[0] belongs to chunks[0]
vectors[1] belongs to chunks[1]
...


So the pack format is:

pack metadata
+ chunk text/metadata
+ chunk embeddings
